# GameBox

GameBox is a small collection of mini games built with **React.js**.

The project is intended as a learning and showcase platform, featuring classic games implemented with modern frontend technologies such as React and responsive design.

## Included Games
- Tic Tac Toe
- Pong
- 2048
- Sudoku

## Tech Stack
- React.js
- JavaScript (ES6+)
- CSS (Responsive Design)
- Docker & Docker Compose (for deployment)

## Deployment
The project can be built and deployed using Docker.  
A production-ready Docker image is served via **Nginx**.

## Goal
The main goal of GameBox is to practice frontend development, game logic, and deployment workflows using Docker.
